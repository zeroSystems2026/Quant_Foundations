# Scratch Project Alpha: Synchronized Loop

* **Initialization:** Reset 'MarketData' to 0.
* **The Bear:** Loops forever; says "Hello!", then broadcasts 'StartAction' and waits.
* **The Fish:** Receives signal, glides across screen, plays sound, and teleports back to start.
* **State Change:** 'MarketData' increases by 1 every time the cycle completes.
